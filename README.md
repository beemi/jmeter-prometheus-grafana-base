# jmeter-prometheus-grafana-base
Prometheus &amp; Grafana setup ready for JMeter


## Start the both services.

```docker-compose up -d```

```docker ps```

```docker stop prometheus```

## Kill the containers

```docker-compose down```

Access Grafana from browser

http://localhost:3000/

Check Prometheus

http://localhost:9090/graph

http://localhost:9270/metrics
